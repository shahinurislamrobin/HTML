HTML 5 tag list

Tag Description
<!DOCTYPE> Declares the document type and version of HTML being used.
<a> Defines a hyperlink.
<abbr> Defines an abbreviation or acronym.
<address> Defines contact information for an author or owner of a document.
<area> Defines an area inside an image map.
<article> Represents a self-contained content block.
<aside> Defines content that is tangentially related to the main content.
<audio> Embeds sound content.
<b> Specifies bold text, without implying added importance.
<base> Specifies the base URL or target for all relative URLs in a document.
<bdi> Isolates a portion of text for bi-directional text formatting.
<bdo> Overrides the current text direction.
<blockquote> Defines a section quoted from another source.
<body> Defines the document body.
<br> Inserts a single line break.
<button> Defines a clickable button.
<canvas> Used to draw graphics via scripting (e.g., JavaScript).
<caption> Defines a table caption.
<cite> Defines the title of a work.
<code> Specifies a piece of computer code.
<col> Specifies column properties for an HTML table.
<colgroup> Defines a group of columns in a table for formatting.
<data> Links content with a machine-readable value.
<datalist> Specifies a list of predefined options for an input element.
<dd> Defines a description or value in a description list.
<del> Specifies text that has been deleted from a document.
<details> Defines additional details the user can view or hide.
<dfn> Specifies a term that is being defined.
<dialog> Defines a dialog box or interactive window.
<div> Defines a generic container for content.
<dl> Defines a description list.
<dt> Defines a term in a description list.
<em> Marks text with emphasis.
<embed> Embeds external content, such as a plugin.
<fieldset> Groups related elements in a form.
<figcaption> Defines a caption for a <figure> element.
<figure> Represents self-contained content, often with a caption.
<footer> Defines the footer of a document or section.
<form> Defines an HTML form for user input.
<h1> to <h6> Define headings from most important (<h1>) to least important (<h6>).
<head> Contains metadata and links for scripts, styles, etc.
<header> Defines the header of a document or section.
<hgroup> Groups heading elements.
<hr> Defines a thematic break in the content.
<html> Defines the root of an HTML document.
<i> Specifies italic text, often for alternate voice or technical terms.
<iframe> Embeds another HTML page within the current one.
<img> Embeds an image.
<input> Defines an input field for user data.
<ins> Specifies text that has been inserted into a document.
<kbd> Represents keyboard input.
<label> Defines a label for an <input> element.
<legend> Provides a caption for a <fieldset>.
<li> Defines a list item.
<link> Defines a relationship between the document and an external resource.
<main> Specifies the primary content of a document.
<map> Defines a client-side image map.
<mark> Highlights text.
<meta> Defines metadata about the document.
<meter> Displays a scalar measurement within a known range.
<nav> Defines a section of navigation links.
<noscript> Defines content to display if scripts are not supported.
<object> Embeds an external resource, such as multimedia.
<ol> Defines an ordered list.
<optgroup> Groups options in a dropdown list.
<option> Defines an option in a dropdown list.
<output> Displays the result of a calculation.
<p> Defines a paragraph.
<picture> Provides multiple image resources for responsive design.
<pre> Defines preformatted text.
<progress> Represents the progress of a task.
<q> Defines a short inline quotation.
<rp> Provides a fallback for browsers that do not support ruby annotations.
<rt> Defines the pronunciation of characters in ruby annotations.
<ruby> Represents ruby annotations for East Asian typography.
<s> Specifies text that is no longer accurate or relevant.
<samp> Defines sample output from a program or system.
<script> Embeds or links to executable code (e.g., JavaScript).
<section> Defines a section in a document.
<select> Creates a dropdown list.
<small> Specifies smaller text.
<source> Defines multiple media resources for elements like <audio> or <video>.
<span> Defines a generic inline container for content.
<strong> Indicates strong importance.
<style> Embeds CSS styles within the document.
<sub> Specifies subscript text.
<summary> Provides a summary for the <details> element.
<sup> Specifies superscript text.
<svg> Embeds Scalable Vector Graphics.
<table> Defines a table.
<tbody> Groups the body content in a table.
<td> Defines a table cell.
<template> Defines a container for reusable content fragments.
<textarea> Defines a multi-line text input field.
<tfoot> Groups footer content in a table.
<th> Defines a table header cell.
<thead> Groups header content in a table.
<time> Defines a specific time or duration.
<title> Specifies the document title.
<tr> Defines a table row.
<track> Defines text tracks for <video> or <audio> elements.
<u> Represents text with an unarticulated annotation, usually styled as underlined.
<ul> Defines an unordered list.
<var> Represents a variable in a mathematical expression or programming context.
<video> Embeds video content.
<wbr> Suggests a line break opportunity.
